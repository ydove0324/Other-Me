# 莫奈睡莲风格 UI 设计规范

> 基于 PRD 的细化设计，仅涉及视觉呈现，不改变交互逻辑。

## 一、莫奈画风参考

- **光影与色彩**：柔和、朦胧，无硬朗轮廓
- **色调**：蓝绿主调 + 粉紫点缀，冷暖交织
- **质感**：印象派笔触感，水波、倒影、模糊边界

## 二、全局设计系统

### 2.1 色彩系统（低饱和度）

| 名称       | Hex       | 用途                     |
|------------|-----------|--------------------------|
| 奶油白     | `#f0eec8` | 主背景、高亮             |
| 莫兰迪绿   | `#80d2af` | 主色、睡莲叶、进度条     |
| 雾霾蓝     | `#7b9e9e` | 次色、水面、边框         |
| 藕粉       | `#dd89c4` / `#e8b4b8` | 点缀、睡莲花瓣   |
| 深绿       | `#103811` | 深色文字、强调           |
| 钴蓝       | `#4e73d6` | 链接、按钮（替换原 indigo）|
| 米色       | `#e8e4d8` | 画布底色                 |

### 2.2 字体

- **标题**：Noto Serif SC（衬线，手写感）
- **正文**：Noto Serif SC 或 Noto Sans SC（稍柔和）
- 避免过于数码感的无衬线体

### 2.3 质感

- 背景： subtle 画布肌理（CSS `background-image` 或 SVG 噪点）
- 圆角：偏大（`rounded-2xl` / `rounded-3xl`）
- 阴影：柔和、偏暖色

## 三、各页面设计要点

### 3.1 Landing（首页）

- 背景：渐变 `奶油白 → 雾霾蓝`，叠加轻微画布噪点
- 标题：衬线体，主色用莫兰迪绿或深绿
- 按钮：圆角气泡感，莫兰迪绿/雾霾蓝，hover 轻微涟漪感

### 3.2 Onboarding 进度条

- 底色：雾霾蓝 10% 透明度
- 进度：莫兰迪绿，边缘略模糊（可选）
- 步骤文字：雾霾蓝、藕粉点缀

### 3.3 选择题页（BubbleQuiz + BubbleQuestion）

- 整体：米色/奶油白背景，画布质感
- **选项样式**：透明气泡感
  - 未选：半透明描边，内部淡淡睡莲纹理（径向渐变模拟）
  - 选中：莫兰迪绿/藕粉填充，轻微光晕
- 按钮：圆角气泡，避免硬朗矩形

### 3.4 人生岔路口（ForkPointInput）

- 标题：衬线体，深绿/雾霾蓝
- 表单：米色背景输入框，雾霾蓝 focus 环
- 实际选择 / 未走的路：用不同莲色区分（绿 vs 粉）

### 3.5 另一个我（StoryView）

- 背景：莫奈睡莲局部感——蓝绿渐变 + 水面波纹（CSS 动画）
- 故事文字：浮在水面上（白/奶油色底 + 轻微阴影）
- 加载态：涟漪/水波动画
- 按钮：莲叶/花瓣色系

### 3.6 Dashboard / CreateFork

- 统一莫奈色系，卡片用奶油白 + 雾霾蓝描边
- 状态标签：藕粉（已完成）、雾霾蓝（进行中）

## 四、实现说明

- 仅修改 `className`、`style`、CSS 变量、Tailwind 配置
- 不修改 `onClick`、`onChange`、`useState`、API 调用等交互逻辑
- 气泡“涟漪”“鱼游”等动效：优先用 CSS 动画实现，避免改变现有状态流
